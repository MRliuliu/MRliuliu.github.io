tcpdump常用用法

from：https://www.cnblogs.com/ggjucheng/archive/2012/01/14/2322659.html
tcpdump的使用方法。

指定网络接口

tcpdump -i iface-name

指定所有网络接口

tcpdump -i any

监听指定主机的数据包

所有进入或者离开主机的数据包(以主机名方式指定主机)
tcpdump -i any host sundown
抓取指定主机之间的通信数据(以主机名方式指定主机)
tcpdump -i any host sundown and \( hel or destopZTE\)
所有进入或者离开主机的数据包(以IP方式指定主机)
tcpdump -i any host 192.168.1.167
抓取指定主机之间的通信数据(以IP方式指定主机)
tcpdump -i any host 192.168.1.167 and \( 172.20.1.122 or 192.168.1.88\)
抓取出指定主机外的通信包
tcpdump -i any host 210.1.1.87 and ! 172.20.1.189
抓取指定主机作为源发送的数据包
tcpdump -i any src host DESKTOP-HPCK5K8
抓取所有目标地址为指定主机的数据包
tcpdump -i any dst host DESKTOP-HPCK5K8

通过指定主机和端口监听数据包

加入tcp端口的筛选
tcpdump -i any host 192.168.1.22 and tcp port 23
加入udp端口的筛选
tcpdump -i any udp port 123 src 192.168.2.111

指定协议抓取数据包

抓取协议
tcpdump -i any 'tcp[tcpflags] & (tcp-syn|tcp-fin) != 0'

抓取包保存

tcpdump -i any -w file.pcapng
