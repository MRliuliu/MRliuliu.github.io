变量与常量

变量

float： 单精度浮点值。1位符号位，8位指数，23位小数。

float可以保证十进制科学计数法小数点后6位有效精度和第7位的部分精度

双精度：1位符号，11位指数，52位小数。

double可以保证十进制科学计数法小数点后15位有效精度和第16位的部分精度。

存放有小数点的数

整数部分转换成二进制采用除2取余的方法。

小数部分采取乘2取整数的方法。

所以22.45表示二进制为：10110.011100110 = 1.0110011100110 x 2^4
然后1可以省去

精度

float和double的精度是由尾数的位数来决定的。浮点数在内存中是按科学计数法来存储的，其整数部分始终是一个隐含着的1，由于它是不变的，故不能对精度造成影响。

float：2^23 = 8388608，一共七位，这意味着最多能有7位有效数字，但绝对能保证的为6位，也即float的精度为6~7位有效数字；

double：2^52 = 4503599627370496，一共16位，同理，double的精度为15~16位。

左值、右值

左值（lvalue）：指向内存位置的表达式被称为左值（lvalue）表达式。左值可以出现在赋值号的左边或右边。
右值（rvalue）：术语右值（rvalue）指的是存储在内存中某些地址的数值。右值是不能对其进行赋值的表达式，也就是说，右值可以出现在赋值号的右边，但不能出现在赋值号的左边。

存储类

auto存储类

auto 只能用在函数内，即 auto 只能修饰局部变量。

register存储类

register 存储类用于定义存储在寄存器中而不是 RAM 中的局部变量。这意味着变量的最大尺寸等于寄存器的大小（一般小于或等于整型长度），且不能对它应用一元的 ‘&’ 运算符（因为它没有内存位置）。

static存储类

被修饰的变量程序生命周期内,但是作用域被限制在文件内部。

extern存储类

使用 extern 时，对于无法初始化的变量，会把变量名指向一个之前定义过的存储位置。但是会是变量对所有程序的文件都是可见的。
