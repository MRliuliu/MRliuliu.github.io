c基础

发表于 2020-12-16 更新于 2021-01-15 分类于 programing ， Clang 阅读次数：

from: https://www.runoob.com/cprogramming/c-tutorial.html
C 语言是一种通用的、面向过程式的计算机程序设计语言。1972 年，为了移植与开发 UNIX 操作系统，丹尼斯里奇在贝尔电话实验室设计开发了 C 语言。
当前最新的 C 语言标准为 C18 ，在它之前的 C 语言标准有 C17、C11…C99 等。

结构

C程序包含预处理器指令、函数、变量、语句&表达式、注释。

令牌

C程序由各种令牌组成，令牌可以是关键字、标识符、常量、字符串值，或者是一个符号。

使用频率较少的关键字

  • const: 定义常量，使用const后变量不能修改。
  • register：声明寄存器变量，这种变量没有地址，无法使用&取地址；但是在内存中存在副本，可以打印出副本地址。
  • volatile：使变量不会被编译器优化，直接读取内存中的变量值，不直接读缓存。

C99新增关键字

  • _Bool: C99的布尔类型，长度为1。
  • _Complex：复数类型
  • _Imaginary：复数类型
  • inline： 定义一个内联函数，主要用来替代C中表达式形式的宏定义。
  • restrict：只用于限定指针，告诉编译器，所有修改该指针指向内容的操作全部是基于该指针的，不存在其他进行修改操作的途径，有助于编译器优化。

C11新增关键字

  • _Alignas：变量内存的控制方式。
  • _Alignof：获取指定表达式或类型的对齐。
  • _Atomic：原子。
  • _Generic：泛型(下方例子)。
  • _Noreturn：在函数返回值类型前面表示函数无返回值。
  • _Static_assert：静态断言，在编译期间就能判断出问题。
  • _Thread_local：线程本地存储。

C数据类型

  • 基本数据类型
    整型和浮点型
  • 枚举数据类型
    也是算数类型，被用来定义在程序中只能赋予其一定的离散整数值的变量。
  • void类型
    类型说明符，没有可用的值。(包括三种情况函数返回为空、函数参数为空、指针指向void)
  • 派生类型
    包括指针类型、数组类型、结构类型、共用体类型和函数类型。

某些数据类型表示方式

整形数据：
1、十进制：10
2、八进制：045
3、二进制：0b11101101
4、十六进制：0x12bf
浮点类型：
1、单精度常量：2.3f
2、双精度常量：2.3(默认就是双精度)

数据类型转换：

1、数据类型转换
存在不同类型的常量和变量，计算是会转换为同一种类型。
2、自动转换
低类型转换为高类型
右值类型转换为左值类型
传参时，转换为形参的类型
函数返回值转换为返回值类型。
